# Project Roadmap

This document describes the intended evolution of the system beyond the M3 baseline.

## Completed Milestones
-	M0 — Foundations
-	M1 — Event Ingestion
-	M2 — Job Processing
-	M3 — Manual Intervention

## Planned Milestones

### M4 — Domain Semantics
-	Explicit domain-level job meanings
-	Business-oriented state transitions
-	Domain validation rules

### M5 — Automation Policies
-	Controlled retries
-	Backoff strategies
-	Policy-driven automation

### M6 — Operational Tooling
-	Metrics and observability
-	Operator workflows
-	Failure diagnostics

### M7 — Productization
-	API contracts for consumers
-	External-facing guarantees
-	Deployment and scaling considerations

The M3 baseline is preserved as a stable reference point for all subsequent evolution.
